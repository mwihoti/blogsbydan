# Medium Checklist

## Before Publishing to Medium
- [ ] Canonical URL is set to your blog website
- [ ] Article is already published on your own blog
- [ ] Title is under 100 characters
- [ ] Subtitle is under 140 characters
- [ ] 3-5 tags selected from Medium's allowed list
- [ ] Cover image is uploaded and credited
- [ ] Image alt text is descriptive
- [ ] No broken Medium-specific formatting
- [ ] Code blocks use Medium-compatible syntax

## Medium-Ready Export
- [ ] Front matter includes `canonical_url`
- [ ] All images have absolute URLs (or uploaded separately)
- [ ] Footnotes converted to inline links
- [ ] No Obsidian-specific syntax remains
- [ ] Medium topics are listed in metadata

## After Publishing
- [ ] `medium_url` saved into post front matter
- [ ] Post moved or copied to `content/published/`
- [ ] Status updated to `published` in front matter
